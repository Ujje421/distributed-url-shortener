# Distributed URL Shortener

## Overview
A highly scalable, distributed URL shortener service.

## Tech Stack
- **FastAPI** - High-performance Python web framework
- **Redis** - High-speed caching for fast redirects
- **PostgreSQL** - Persistent storage for URLs and metadata
- **Docker** - Containerization for consistent environments
- **CI/CD** - Automated testing and deployment
- **Metrics & Observability** - Monitoring system performance

## Key Features
- Generate short URLs from long ones.
- Redirect to original URLs with minimal latency.
- Handle high throughput with caching.
- Capture analytics and click metrics.
